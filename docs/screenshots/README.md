# Screenshots

このフォルダにプラグインのスクリーンショットを置いてください。
README.md から参照されます。

## 必要な画像

- `main.png` — メイン画面のスクリーンショット
- (任意) `kick-tab.png` — KICKタブ
- (任意) `bass-tab.png` — BASSタブ
- (任意) `in-logic.png` — Logic Pro で読み込まれている様子

## 撮影のコツ

- macOS で **Shift + Cmd + 4** → スペースキー → ウィンドウクリックで
  影付きのきれいなスクリーンショットが撮れます
- ファイルサイズは 500KB 以下を目安に (PNG なら 1200x900 で十分)
- 不要な背景・他のアプリが映り込まないように
